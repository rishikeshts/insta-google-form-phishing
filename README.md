# 🛡️ Cyber Awareness Education Tool

A high-fidelity, premium Instagram-inspired login interface designed for conducting **cyber awareness sessions for children**. This project aims to educate young users about online safety, identifying phishing attempts, and responsible internet usage through a realistic, interactive demonstration.

---

## ✨ Features

- **Premium Design Aesthetics**: Vibrant colors, glassmorphism, and dynamic animations for a modern, high-end feel.
- **Micro-Animations**: Subtle hover effects, floating mockups, and entrance animations to engage users.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile viewing.
- **Real-time Data Collection**: Demonstrates how data can be captured using integrated Google Forms as a mock backend.
- **Interactive Popups**: Includes "Logged Out" and "Incorrect Password" demonstrations.

---

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/rishikeshts/insta-google-form-phishing.git
    cd insta-google-form-phishing
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm start
    ```
    Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## 🛠️ Built With

- **React**: For building the component-based user interface.
- **Vanilla CSS**: Custom styling with modern CSS features (gradients, animations, glassmorphism).
- **Lucide React**: For sleek, lightweight iconography.
- **Google Forms**: Used as a demonstration backend for data collection.

## 📝 Educational Purpose & Disclaimer

This project was created solely for **educational purposes** during cyber awareness training. It is designed to help children and young adults understand:
- How to recognize unofficial login pages.
- The importance of checking URLs before entering credentials.
- How data collection works on the web.

> [!CAUTION]
> This project is NOT intended for phishing or any fraudulent activity. Users are encouraged to use it ethically within educational contexts only.
